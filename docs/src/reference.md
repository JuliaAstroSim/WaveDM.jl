## JuliaAstroSim Ecosystem

WaveDM.jl is part of the JuliaAstroSim ecosystem, which provides a comprehensive set of tools for astrophysical simulations:

1. **AstroSimBase.jl**: Basic types and interfaces for astrophysical simulations
2. **PhysicalParticles.jl**: N-body particle types and vector algebra
3. **PhysicalFFT.jl**: FFT solvers for physical problems
4. **AstroNbodySim.jl**: Gravitational N-body simulations
5. **AstroIC.jl**: Initial condition generation for astrophysical simulations
6. **AstroPlot.jl**: Visualization tools for astrophysical data
7. **GalacticDynamics.jl**: Density profiles and orbit calculations
